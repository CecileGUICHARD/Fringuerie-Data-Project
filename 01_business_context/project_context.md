# Project Context

La Fringuerie is a second-hand textile activity run by the association Les Échos Ludiques.

This project combines two complementary dimensions:

## 1. Data analysis project
Carried out during my Data Analytics training:

- SQL analysis  
- KPI definition  
- Looker dashboarding  
- Operational recommendations

## 2. Operational data collection initiative
Currently being structured for the association:

- Google Forms data capture  
- Google Sheets structuring  
- Cash register monitoring logic  
- Data quality controls

## Core idea

The project illustrates a complete data approach:

Business need → Data quality → Analysis → Decision support
