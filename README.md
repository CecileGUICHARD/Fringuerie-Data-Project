# Fringuerie Data Project

## Overview

This project focuses on the data analysis and operational tracking of a local second-hand association shop: **La Fringuerie**, run by the non-profit organization *Les Échos Ludiques*.

The project combines two complementary dimensions:

1. A **data analysis project** carried out during my Data Analytics training, using SQL, KPI definition and Looker dashboarding.
2. An **operational data collection initiative** currently being structured for the association using Google Forms and Google Sheets.

The objective is to show how data can support a small non-profit activity, from raw operational information to business insights and future decision-making.
