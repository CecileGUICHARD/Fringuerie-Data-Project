# Analysis Summary

This part of the project was carried out during my Data Analytics training.

The objective was to analyze sales data from La Fringuerie in order to better understand the activity and identify useful indicators for decision-making.

The analysis focused on:

- sales performance
- product categories
- quantities sold
- revenue distribution
- payment methods
- operational recommendations

The project helped identify both business insights and data quality improvements needed for future monitoring.

## Main analytical approach

The analysis followed these steps:

1. Understand the business context
2. Explore available data
3. Clean and structure key fields
4. Aggregate data using SQL
5. Define useful KPIs
6. Build visual insights in Looker
7. Translate results into operational recommendations

## Key learning

This project showed that dashboarding is only the final step of a data project.

Before building visuals, it is necessary to:

- understand the business need
- make the data reliable
- choose relevant indicators
- structure the analysis clearly
